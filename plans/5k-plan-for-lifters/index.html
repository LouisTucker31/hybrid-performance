FAQs

Common questions about this plan.

How is this different from a generic 5K plan?

This plan is built around lifting first. Run days and intensity are placed to work with your leg days and recovery, not against them, rather than a generic couch-to-5K schedule that ignores your gym training.

Will running affect my strength gains?

Not if it's structured properly. Two runs a week at the right intensity and placement shouldn't meaningfully interfere with strength progress, that's the whole point of how this plan is built.

What if I've never run before?

That's fine. The plan progresses gradually from week one, so no running background is assumed.

What do I need to use the tracker?

Just a free Google account for Google Sheets, or Microsoft Excel if you prefer. Both versions are included.

Is there support if I get stuck?

This is a self-guided plan without ongoing coaching support. If you'd prefer input along the way, a Bespoke Plan is the better fit.
